# docs/brand — Universo de marca

Fuente original de la identidad visual de Ynara: la **Guía de identidad
visual · Ynara 2026**.

- [`Ynara-Universo-de-Marca.html`](./Ynara-Universo-de-Marca.html) — guía
  completa (esencia de marca, sistema gráfico "Red de memoria", paleta,
  tipografía, iconografía y aplicaciones). Es un documento estático
  autocontenido; abrir en el navegador.

Esta guía es la **fuente del sistema gráfico, la iconografía y la paleta**
que [`DESIGN.md`](../../DESIGN.md) (raíz) traduce a especificación de
implementación. Si hay divergencia entre esta guía y el código, el orden de
verdad es: guía de marca → `DESIGN.md` → `globals.css`.

> Versionada en el repo (antes vivía solo en local) para que el sistema de
> diseño v2 tenga su fuente primaria trazable, como pide `DESIGN.md` §15.
